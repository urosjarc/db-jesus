NamingTests

class NamingTests(val ser: Serializer)

The class is responsible for performing tests on element names. These tests are needed in case developer forgets to use escaping quotation while creating queries in Serializer. Database will create very strange exception messages which are really hard for the user to understand and to figure out where is the source of the problem. That's why we force naming convention of elements that will be used inside the database.
